# Energy Consumption Prediction

This project focuses on predicting energy consumption using Machine Learning techniques. The notebook includes data preprocessing, exploratory data analysis (EDA), visualization, model training, and prediction using advanced regression algorithms.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Data visualization using Matplotlib and Seaborn
- Machine Learning model training
- Energy consumption prediction using XGBoost
- Model evaluation and performance analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- XGBoost
- Jupyter Notebook

## Project Structure
```

Energy_Consumption_Prediction.ipynb
README.md
requirements.txt

```

## Installation

1. Clone the repository:
```bash
git clone <your-repository-link>
```

2. Navigate to the project folder:
```bash
cd energy-consumption-prediction
```

3. Install the required dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the Jupyter Notebook:
```bash
jupyter notebook Energy_Consumption_Prediction.ipynb
```

## Output
The project predicts energy consumption values based on the given dataset and visualizes model performance through graphs and evaluation metrics.
